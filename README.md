# Cara install 

<pre>
[!] Pastikan sudah install library esp8266 dan library DHT11 di arduino ide lu

-- Cara install library esp8266 --
https://www.instructables.com/Quick-Start-to-Nodemcu-ESP8266-on-Arduino-IDE/

-- Cara install library DHT11 -- 
https://create.arduino.cc/projecthub/pibots555/how-to-connect-dht11-sensor-with-arduino-uno-f4d239

</pre>

<pre>
-- Setting Nodemcu --

1. Koneksikan nodemcu ke pc 
2. buka arduino ide
3. open file dan masukan script magerproject.ino
4. compile program nya
![](https://raw.githubusercontent.com/FajarTheGGman/The-Mager-Project/master/.img/ide.png)
5. Buka Serial Monitor dan tuggu sampai ip nodemcu muncul
![](https://raw.githubusercontent.com/FajarTheGGman/The-Mager-Project/master/.img/ipnode.png)
6. Selesai
</pre>
